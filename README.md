# Aprendiendo Python y Programación Orientada a Objetos

Repositorio personal donde documento mi aprendizaje de Python y Programación Orientada a Objetos (POO).

## Objetivo

Adquirir una base sólida en Python antes de continuar con el desarrollo backend utilizando Django.

## Temas estudiados

- Clases y Objetos
- Atributos
- Métodos

## Próximos temas

- Encapsulamiento
- Herencia
- Polimorfismo
- Clases abstractas
- Relaciones entre objetos
- Manejo de excepciones
- Archivos
- Bases de datos
